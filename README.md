## Description

걍 개인 블로그 있는게 멋있어보여서 무작정 시작해보는 내 개인 블로그 프로젝트
보통 공부한 내용들 velog를 사용해서 메모중이었는데, 개발자들 블로그 만드는거 좀 멋있어 보여서 따라해보려고 시작했습니다.
공부하거나 팀플젝하는 중간중간 시간나면 글들 옮겨서 적고 커밋하겠습니다
